# SURDOCS External Frontend

React + TypeScript frontend scaffold created with Vite.

## Stack

- React 18
- TypeScript
- Vite
- Material UI
- React Router
- Axios

## Folder Structure

```text
src/
  components/
  hooks/
  pages/
  routes/
  services/
  theme/
  types/
```

## Run

```bash
npm install
npm run dev
```

The default API base URL points to:

`http://localhost:8080/surdocs-external`
